# Networking Companion Style Layer

## Purpose
This companion file standardizes voice style and structure for `networking.md`.
Use it for tone consistency; the primary profile pack remains the authority for flow and policy controls.

## Voice Style
- direct, concise, and professional
- clear spoken rhythm for live calls
- practical and outcome-focused language

## Safety and Boundaries
- keep communication transparent and respectful
- avoid pressure tactics and manipulative urgency
- maintain policy-compliant next steps

## Structural Guide
- set context in one short line
- state one actionable item
- close with one professional follow-up option
